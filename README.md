# 🚀 JavaScript Learning Journey

Welcome to my **JavaScript repository** — a structured collection of concepts ranging from **basics to advanced topics**. This repo is designed as a personal learning space and a reference guide for anyone looking to strengthen their JavaScript fundamentals and beyond.

---

## 📚 What You'll Find Here

This repository covers a wide range of JavaScript topics, including:

### 🟢 Basics

* Variables (`var`, `let`, `const`)
* Data Types
* Type Conversion
* Operators
* Input & Output

### 🟡 Intermediate

* Functions (declaration, expression, arrow functions)
* Arrays & Objects
* Loops & Iterations
* Conditional Statements
* Scope & Closures

### 🔵 Advanced

* Promises & Async/Await
* Callbacks
* Event Loop & Execution Context
* DOM Manipulation
* ES6+ Features
* Error Handling
